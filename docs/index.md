# EngineeringUnits

A .NET class library for representing physical quantities as first-class typed values, with units carried explicitly and preserved through arithmetic.

## Status

Version 0.1.0-alpha. Phases 1–5 implemented plus Tiers A–D cleanup. 283 tests passing.

The library is consumable as a NuGet package (`EngineeringUnits`) targeting **.NET Standard 2.0**, callable from both .NET Framework 4.7.2+ and modern .NET 6+.
