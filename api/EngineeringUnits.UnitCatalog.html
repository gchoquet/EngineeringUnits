Remarks

Lookup order:

  1. Exact symbol match, case-sensitive (so "m" ≠ "M").
  2. Long-name match, case-insensitive (so "foot", "Foot", "FOOT" all match the unit whose LongName is "foot").
  3. Alias match (irregular plurals and common alternate forms), case-insensitive — e.g. "feet", "inches".

Returns false if nothing matches.
