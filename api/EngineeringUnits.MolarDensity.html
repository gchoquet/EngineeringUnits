Remarks

Common units: mol/m³, mol/L, kmol/m³ (= mol/L numerically), mol/dm³ (= mol/L), lbmol/ft³. Pipeline / chemical-engineering work often quotes density in mol/L because that matches AGA8 / GERG-2008 internal conventions.
