Remarks

Instances are immutable. Subclasses ( Length, Mass, Time, Temperature, …) represent specific dimensions and carry their own typed arithmetic operator overloads.

Internally every quantity is stored in canonical SI base units, with the original display unit retained for formatting and for the left-operand-wins rule on arithmetic results (see specification §14.4).

ToString(string?, IFormatProvider?)

Formats this quantity. Supported format codes (see specification §8.2 and §14.17):

  • G or nullGeneral default — short symbol.
  • LLong unit name (e.g. "feet").
  • DDual — primary value plus SI or US-customary equivalent in parens.
  • SSI canonical (from SIScientific).
  • UUS-customary canonical (from UsCustomary).
  • PPreferred (from Default).
  • EScientific notation.
  • N{n}Force n significant figures, e.g. "N6".
  • {unit}Force a specific unit, e.g. "{m}".
