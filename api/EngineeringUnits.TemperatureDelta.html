Remarks

Examples:

  • 1 °C delta = 1 K delta (same magnitude — both linear in K)
  • 1 °F delta = 1 °R delta = 5/9 K delta (different scale, no offset)

Temperature - Temperature produces a TemperatureDelta. TemperatureDelta + Temperature produces a Temperature (raising the absolute by the delta). TemperatureDelta + TemperatureDelta is itself a TemperatureDelta. Two absolute temperatures cannot be added (throws — see operator +(Temperature, Temperature)).

TemperatureDelta(double, string)

Creates a temperature-delta from a value and the unit symbol of the scale the delta is expressed on. For Celsius and Kelvin deltas use the same symbols as absolute temperatures (numerically equivalent). For Fahrenheit / Rankine deltas the symbol's scale factor (5/9) is honored but the offset is ignored.
