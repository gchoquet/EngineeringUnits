EngineeringUnits
Table of Contents

Namespace EngineeringUnits

Classes

Acceleration

An acceleration quantity. SI base unit: meter per second squared (m/s²).

AmountOfSubstance

An amount of substance. SI base unit: mole.

AngularVelocity

An angular velocity quantity. SI base unit: radian per second (rad/s).

Area

An area quantity. SI base unit: square meter (m²).

AreaDensity

Area density (mass per unit area). SI base unit: kilogram per square-meter (kg/m²).

Density

A mass density quantity (mass per unit volume). SI base unit: kg/m³.

DerivedUnit

A quantity whose dimension does not match any registered subclass. Returned by arithmetic when the result combination has no dedicated type, e.g. Mass * Length / Time^3.

DimensionMismatchException

Thrown when an operation requires two quantities of the same dimension but they differ (e.g. adding a Length to a Time, or calling As("ft") on a Mass).

DimensionlessQuantity

A quantity with no physical dimension — a pure number. Produced by arithmetic that reduces to all-zero dimensions (e.g. velocity / velocity, length / length).

DynamicViscosity

Dynamic (absolute) viscosity. SI base unit: pascal-second (Pa·s).

ElectricCapacitance

Electric capacitance. SI unit: farad (F = C/V).

ElectricCharge

Electric charge. SI unit: coulomb (C = A·s).

ElectricCurrent

Electric current. SI base unit: ampere (A).

ElectricResistance

Electric resistance. SI unit: ohm (Ω = V/A).

ElectricalConductance

Electrical conductance (reciprocal of resistance). SI unit: siemens (S = 1/Ω).

Energy

An energy / work quantity. SI base unit: joule (J).

EnergyDensity

Energy density (energy per unit volume). SI base unit: joule per cubic-meter (J/m³).

EnergyPerArea

Energy per unit area (e.g. fracture toughness E/A, surface energy). SI base unit: joule per square meter (J/m²). Dimensionally equal to SurfaceTension (M/T²) but semantically distinct — kept separate for the same reason Torque and Energy are separate despite sharing M·L²/T².

EngineeringUnit

Abstract base class for all physical-quantity types. Carries a value expressed in canonical SI base units, a display unit, and significant-figure metadata.

EngineeringUnitException

Base class for all exceptions thrown by this library.

Force

A force quantity. SI base unit: newton (N).

Frequency

A frequency quantity. SI base unit: hertz (Hz = 1/s).

HeatCapacity

Heat capacity (per object, not per mass). SI base unit: joule per kelvin (J/K).

HeatFluxDensity

Heat flux density (power per unit area). SI base unit: watt per square-meter (W/m²).

Inductance

Electrical inductance. SI unit: henry (H = V·s/A = Wb/A).

KinematicViscosity

Kinematic viscosity (dynamic viscosity divided by density). SI base unit: square-meter per second (m²/s).

Length

A length quantity. SI base unit: meter.

Mass

A mass quantity. SI base unit: kilogram.

MassFlowRate

A mass flow rate (mass per unit time). SI base unit: kg/s.

MixtureMolarConversions

Bridges molar-basis quantities (e.g. MolarEnergy in J/mol) to mass-basis quantities (e.g. SpecificEnergy in J/kg) and back, using a mixture's MolarMass.

MolarDensity

A molar density quantity. SI canonical: mol/m³ (dimension N/L³).

MolarEnergy

A molar energy quantity. SI canonical unit: J/mol (dimension L²·M/(T²·N)).

MolarHeatCapacity

A molar heat-capacity (or molar entropy) quantity. SI canonical: J/(mol·K).

MolarMass

A molar mass quantity. SI canonical unit: kg/mol (dimension M/N).

Momentum

Linear momentum (mass × velocity). SI base unit: kilogram-meter per second (kg·m/s).

PlaneAngle

A plane angle quantity. SI base unit: radian (rad).

Power

A power quantity. SI base unit: watt (W).

Pressure

A pressure quantity. SI base unit: pascal (Pa).

SpecificEnergy

Specific energy (energy per unit mass). SI base unit: joule per kilogram (J/kg).

SpecificHeatCapacity

Specific heat capacity (per unit mass). SI base unit: joule per kilogram-kelvin (J/(kg·K)).

SpecificVolume

Specific volume (volume per unit mass — reciprocal of density). SI base unit: cubic-meter per kilogram (m³/kg).

SurfaceTension

Surface tension (force per unit length). SI base unit: newton per meter (N/m).

Temperature

An absolute temperature. SI base unit: kelvin.

TemperatureDelta

A temperature difference (delta). Distinct from Temperature because temperature differences obey linear-only conversion (no offset), while absolute temperatures use affine conversion.

ThermalConductivity

Thermal conductivity. SI base unit: watt per meter-kelvin (W/(m·K)).

Time

A time-duration quantity. SI base unit: second.

Torque

A torque (moment of force) quantity. SI base unit: newton-meter (N·m).

UnitCatalog

The library's catalog of recognized units. Populated once at module load and then frozen — concurrent reads are safe without locking.

UnitParseException

Thrown when a value+unit string cannot be parsed.

UnitPreferences

Per-dimension display-unit preferences plus notation style. Used by the "P" format code and by ToDualString().

UnknownUnitException

Thrown when a unit symbol is not registered in the catalog.

Velocity

A velocity (linear speed) quantity. SI base unit: meter per second (m/s).

Voltage

Electrical potential difference (voltage). SI unit: volt (V = W/A = J/C).

Volume

A volume quantity. SI base unit: cubic meter (m³).

VolumetricFlowRate

A volumetric flow rate (volume per unit time). SI base unit: m³/s.

Structs

DimensionSignature

The physical dimension of a quantity, expressed as a signed exponent for each SI base dimension plus a pseudo-dimension for plane angle (radians).

Unit

A unit of measurement: a name, a dimension, and conversion factors to/from the canonical SI-base value.
